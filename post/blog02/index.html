    • 基于Hugo的个人博客搭建(mac)

    Wed, May 20, 2020 1-minute read

    博客是个人学习总结的利器，尤其是互联网知识。以前试着在CSDN写过一些总结，但肯定没有个人博客来的炫酷😝

    每次百度技术性的问题，前几名不再是百度知道而是各种博客的总结。现在互联网的发展速度、知识多样化、碎片化，如果不及时进行总结，过一阵还是会去百度问题答案的。

    “对任何渴望进步的人来说，写博客/文章/回答对自己的成长帮助都是巨大的。频率和数量不必过分强求，有话要说就写写，没事就歇歇，随意点。因为坚持写作的那一刻起，你就已经开始受益。”

    此篇博客搭建根据B站up「CodeSheep」教程

    采用Hugo框架下的anatole模板，部署于GitHub

    • ## 前提准备

    安装有 git、brew

    • ## 下载Hugo

    这里Up主采用的方式是brew命令下载，如果是win或网络不好下不来，可以去github找hugo自行下载对应版本

    brew install hugo

    下载完后输入hugo version，正常显示版本则表示安装成功

    • ## 建立我的blog站点

    执行命令建立我的blog站点（mylog为文件夹名称，可自行修改）

    hugo new site myblog
    • ## 下载自己喜欢的hugo模板

    首先进入我的博客的根目录

    cd myblog/

    进入hugo主题，选择一款自己喜欢的模板。

    我选择是anotole，没有up主那种快捷下载命令，所以选择download，会跳转github页面，选择Clone or download，复制url链接

    再用git克隆到本地（这步我执行了很久，和网络有关）

    git clone https://github.com/lxndrblz/anatole.git

    下载完后可以进入themes文件夹，查看是否有自己下载的主题，有即为成功。

    • ## 建立本地服务器

    首先回到博客的根目录，执行以下命令

    hugo server -t anatole --buildDrafts

    成功地话这里会出现

    Web Server is available at http://localhost:1313/ (bind address 127.0.0.1)

    这时就可以复制链接http://localhost:1313/到浏览器看一下是否成功了。

    • ## 新建第一篇博客

    本地部署好，可以建立我的第一篇博客了，命令如下：

    hugo new post/blog.md

    现在content/post/文件夹下应该会出现一个名为blog的md文件，现在就可以使用其他编辑器打开写自己的博客了。

    • ## 部署到github服务器

    首先到github新建一个仓库，名字必须是和自己用户名一样，且是全小写+.github.io后缀。

     hugo --theme=anatole --baseUrl="https://zhenggengqiong.github.io/" --buildDrafts

    这里的url地址换成自己的地址，theme主题也换成你下载的主题名字。

    这时博客根目录会出现一个新的public文件夹，现在要把这个文件夹部署到github，分别执行以下命令。

    git init

    git add .

    git commit -m "我的hogo博客第一次提交"
    • ## 推到远端服务器

    首先要和我们的github关联

    git remote add origin https://github.com/zhenggengqiong/zhenggengqiong.github.io.git

    再执行命令推到远端，这样就可以用网址访问博客了。

    git push -u origin master

    过程中会提示输入github的用户名及密码

    成功之后，就可以用你自己的网址访问博客啦。https://zhenggengqiong.github.io/
